## Money API

Money API that allows register all personal transactions ( supermarker, emp, ... )

### Developer Environment 

#### Software Dependencies

- NodeJS v12
- Docker 20.10.0
- Docker Compose 1.27.4

#### Get Started

Start Database
```sh
$ yarn db
```

Start API
```sh
$ yarn install
$ yarn start
```

### Production Environment

```sh
yarn build
```
